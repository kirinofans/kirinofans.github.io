[论文笔记] Chat2SVG: 大语言模型与扩散模型协同的矢量图形生成

paper:Chat2SVG: Vector Graphics Generation with Large Language Models and Image Diffusion Models
Comments:Project Page: this https URL
Subjects:Computer Vision and Pattern Recognition (cs.CV); Graphics (cs.GR)

摘要

Scalable Vector Graphics (SVG) 作为矢量图形标准，具备分辨率无关、可精准控制单个元素的优势，是现代数字设计的核心格式。然而，高质量 SVG 创作门槛高，非专业用户难以掌握。

本文提出 Chat2SVG，一种混合架构，融合了大语言模型（LLM）和图像扩散模型的优势。该方法支持自然语言迭代编辑 SVG，实现了高质量文本到 SVG 的生成。通过 LLM 生成语义模板与扩散模型增强细节的结合，Chat2SVG 同时保证了生成结果的视觉保真度、路径规整性和语义对齐性，解决了现有方法路径碎片化或几何细节缺失的问题。

1. 研究背景与挑战

1.1 SVG 的优势与创作困境

SVG 作为矢量图形标准，具有分辨率无关、可精准控制单个元素的优势。但高质量 SVG 创作门槛高，非专业用户缺乏专业软件操作能力，且复杂形状制作耗时久。

1.2 现有方法的局限性

现有文本转 SVG 方法主要分为两类，但均存在明显缺陷：

  1. 图像基方法：
    • 依赖可微光栅化器将矢量渲染为像素图进行特征匹配。
    • 优点：视觉表达能力强，泛化性好。
    • 缺点：生成的路径易碎片化，单个语义组件被拆分到多个重叠路径，且易出现路径自相交、冗余，缺乏语义对应。
  2. 语言基方法：
    • 将 SVG 代码编码为 token，训练自回归 Transformer 进行生成。
    • 优点：路径规整，语义清晰。
    • 缺点：泛化能力差，受限于领域数据集，LLM 仅能生成基础几何形状，缺乏复杂几何细节。

      核心痛点：现有方法无法同时兼顾 SVG 的路径规整性、语义一致性和视觉表达性。

2. 相关工作

2.1 图像基生成方法

早期方法如 SketchRNN（RNN+VAE）、DeepSVG（Transformer）将整个 SVG 编码为一个单一的潜向量。

  • 问题：对于多部件图形（如卡通狮子），单一潜向量无法承载所有部件的精细几何信息，解码后细节丢失严重。
  • 改进：路径级表示方法为每个部件单独训练 VAE，生成对应的路径级潜向量，保证了路径的平滑性。

2.2 语言基生成方法

如 CLIPDraw，通过文本-图像编码器进行优化。虽然无需针对矢量图形额外训练，但生成的路径往往扭曲杂乱，文本对齐效果较差。

2.3 Chat2SVG 的融合思路

Chat2SVG 融合了两类方法的优势：

  1. LLM 生成模板：生成语义清晰、路径独立的 SVG 骨架（解决路径碎片化）。
  2. 扩散模型补充细节：将简单模板渲染并增强细节，生成参考图（解决 LLM 细节缺失）。

3. 方法

该方法主要分为两个阶段：SVG 模板生成与双阶段优化。

3.1 第一阶段：SVG 模板生成

利用 LLM（Claude-3.5-Sonnet）将自然语言转换为 SVG 代码，并进行视觉校正。

3.1.1 三级提示词扩展

为了保证语义完整性，采用三级扩展策略：

  1. 场景级扩展：分析核心物体，补充配套元素。
  2. 物体级扩展：将物体拆分为最小语义部件（如狮子的身体、头部、鬃毛、四肢等）。
  3. 布局级扩展：指定每个部件的位置、大小、颜色和空间关系。

3.1.2 视觉校正

纯文本生成的 SVG 易出现部件错位、比例失调。

  • 流程：渲染 SVG -> LLM 视觉检查 -> 修正代码。
  • 效果：仅需 2 次迭代即可生成布局规整的 SVG 骨架模板。

3.2 第二阶段：细节增强与新部件挖掘

利用图像扩散模型增强视觉细节。

  1. 细节增强：采用 SDEdit + ControlNet。
    • SDEdit：对骨架像素图加噪后去噪，生成丰富细节。
    • ControlNet：使用高斯模糊的骨架图作为控制信号，保留整体结构和布局。
  2. 新部件挖掘：
    • 扩散模型生成的目标图可能包含模板中没有的装饰部件（如胡子）。
    • 使用 SAM 分割目标图与模板图，识别新部件掩码，转换为多边形路径并添加到 SVG 中。

3.3 核心创新：双阶段 SVG 优化

此阶段旨在让 SVG 图像向目标图对齐，同时解决传统方法的路径自相交、锯齿问题。

3.3.1 第一子阶段：潜空间优化

目标：保证路径规整性与布局一致性。

  • 方法：不直接优化控制点，而是优化预训练 SVG VAE 的潜向量 zzz。
  • 逆变换：由于模板路径曲线数量不固定，通过最小化 EMD（推土机距离）损失，将不规则路径映射到符合 VAE 要求的潜向量。

    • YYY：从模板路径采样的点集。
    • 优化 zzz 使 XXX 匹配 YYY。
  • 损失函数：

    • ℓcurvature\ell_{curvature}ℓcurvature​：减少锯齿和自相交。

3.3.2 第二子阶段：点级优化

目标：补充精细几何细节，解决潜空间优化后形状过于光滑的问题。

  • 方法：将贝塞尔曲线在中点拆分，控制点翻倍，直接优化控制点位置。
  • 策略：移除 IoU 约束，动态调整曲率损失权重 λ3\lambda_3λ3​。
    • 早期 λ3\lambda_3λ3​ 大：优先保证路径顺滑。
    • 后期 λ3\lambda_3λ3​ 小：允许添加精细细节（如鬃毛纹理）。

4. 实验结果与消融实验

4.1 定性对比

  • CLIPDraw / DiffSketcher：生成扭曲杂乱的笔触，文本对齐差。
  • VectorFusion / SVGDreamer：生成锯齿状、碎片化的形状，缺乏语义性。
  • Chat2SVG：生成的 SVG 具有优化的文本对齐、高视觉质量、结构良好的路径和清晰的语义定义。

4.2 消融实验

  1. 移除 SVG 模板：图像 FID 和路径 FID 大幅变差，文本对齐度下降。
  2. 移除双阶段优化：视觉细节缺失，证明双阶段优化是补充精细细节的关键。

4.3 迭代编辑验证

用户可通过多轮指令逐步修改 SVG。系统仅对修改的路径重新优化，保留未修改部分，保证编辑前后视觉一致、语义连贯。

5. 结论与局限

5.1 结论

Chat2SVG 在视觉保真度、路径规整性、文本对齐度三项核心指标上均显著优于基线方法。生成的 SVG 既符合专业设计规范，又能精准匹配文本提示语义，具备丰富的视觉细节，未来有望应用于关键帧动画生成等领域。

5.2 局限性与改进方向

  1. 布局精度不足：LLM 生成的布局可能存在比例失调、位置偏差。
  2. 小部件易遗漏：依赖 SAM 挖掘新部件时，可能遗漏细小装饰元素（如甜甜圈的糖霜）。
  3. 复杂形状表达受限：预训练 SVG VAE 要求固定数量的贝塞尔曲线命令，难以刻画超精细形状（如复杂鬃毛纹理）。
