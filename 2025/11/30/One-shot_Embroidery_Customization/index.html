[论文阅读]One-shot Embroidery Customization via Contrastive LoRA Modulation

[论文笔记] 单样本刺绣定制：让AI学会穿针引线

paper:One-shot Embroidery Customization via Contrastive LoRA Modulation
Comments: Accepted to ACM Transactions on Graphics (TOG), SIGGRAPH Asia 2025
Subjects: Graphics (cs.GR); Computer Vision and Pattern Recognition (cs.CV)

摘要

扩散模型虽然在图像生成领域取得了巨大进展，但在面对“精细视觉特征迁移”任务——如刺绣的针脚、材料纹理生成时，效果往往不尽人意。刺绣作为一种复杂的纺织艺术，其风格核心在于高频结构纹理，而非简单的颜色。现有方法常将颜色误认为是风格，导致风格与内容无法彻底解耦。

本文提出了一种基于对比LoRA调制的单样本刺绣定制方法。通过构建成对数据、筛选SDXL关键模块以及两阶段对比学习训练，首次实现了仅需1张参考刺绣图，即可精准捕捉其精细风格并生成新刺绣。该方法不仅解决了刺绣定制难题，还可泛化至艺术风格迁移、线稿上色等领域。

1. 研究背景与目标

1.1 刺绣定制的独特挑战

扩散模型生成的高保真图片正在改变零售行业流程，尤其是“先卖后产”的预售可视化场景。然而，刺绣定制面临独特挑战：

  1. 风格特殊性：刺绣是复杂纺织艺术，核心是多样针脚和材料（亮片、绒线）的精细组合，与普通艺术风格完全不同。
  2. 数据稀缺：预训练模型无法适配未见过的刺绣风格，而大规模数据集微调因“刺绣数据少、图案差异大”而行不通。
  3. 风格与内容混淆：现有方法常把“颜色”当风格核心，但刺绣的风格是“高频结构纹理”（针脚/材料），导致风格与内容（图案形状）分不开。

1.2 研究目标


本文旨在实现单样本刺绣定制：

  • 输入：文字/图像 + 1张参考刺绣图。
  • 输出：新刺绣图。
  • 核心要求：生成的新刺绣要完全保留输入内容，同时精准复刻参考图的风格（针脚、材料纹理）。
  • 泛化性：方法不仅能做刺绣定制，还能泛化到艺术风格迁移、线稿上色、外观迁移等任务。

2. 相关工作

2.0 What is LORA？

可以参考下面的视频：
LoRA微调从原理到调参，7 个问题彻底理解LoRA，不懂线性代数也没问题_大模型微调_低秩适配

2.1 传统与几何方法

  • 专业CAD软件（如Wilcom）：人工将数字图画转换为机器针脚指令。缺点是全靠手动操作费时间，且生成效果图不够逼真。
  • 早期3D几何建模：模拟基础针脚（长短针、缎面针）的几何形状。缺点是只能做简单针脚，无法处理复杂材料（亮片、绒线），且需手动定义位置。
  • 矢量场/针脚基元模拟：用矢量场模拟随机针脚，多层渲染增强真实感。

2.2 深度学习方法

  • 内在图像分解法：基于图像分割和UV映射。
  • 深度学习/对抗学习：直接生成类刺绣图像。

本文优势：相比上述方法，本文在数据依赖、操作成本、灵活性及实用性上均有显著提升。

3. 核心方法

本文提出的方法主要包含三个核心步骤：成对数据构建、SDXL模块分析、两阶段EmoLoRA训练。

3.1 Pair-wise Data Construction（成对数据构建）

目标：给1张参考刺绣图生成1张对应的“设计图”。这对图的差别即为“风格”，以此明确“学的风格是针脚，保的内容是形状颜色”。

流程（4步生成设计图）：

  1. 边缘检测与形状保持：用 “边缘检测（HED 检测）”和“保形状工具（ControlNet-Canny）”先让 HED 检测器给刺绣图画轮廓线，得到一张 边缘图；
    ControlNet-Canny会把针脚、毛茸茸的纹理全去掉。

  2. 细节补全与颜色对齐：用 ControlNet-Tile
    ControlNet Tile 是 ControlNet 1.1 中的一种模型，专注于图像的细节补全和增强。它通过对输入图像进行条件控制，能够忽略原始图像中的缺陷，并生成更高质量、更精细的图像。该模型不仅适用于细节补全，还可以用于图像放大（超分辨率）和细节微调。
    作用是：让生成的新图和刺绣图 颜色一样，让图的质量更清晰。

  3. 文字描述生成：用 WD14生成描述文字 （WD14-Tagger可以简单理解为一个图生文，反向解析文生图的工具，需要模型文件对图片进行处理）
    给刺绣图自动写一段描述，再给这段描述加上关键词比如 “扁平设计

  4. SD3生成设计图：把前面 3 个工具的结果（边缘、描述词等等）给 SD3；SD3 生成一张和刺绣图 “形状一样、颜色一样，但没有针脚 / 纹理” 的图 —— “设计图”（“Design”）

3.2 Analysis on SDXL（SDXL模块分析）

目标：在 SDXL 预训练扩散模型中，筛选出能够精准捕捉刺绣风格的网络模块，为后续高效训练 LoRA实现风格 - 内容分离提供基础

流程（3步）：

  1. 特征提取：用 ReNoise（一种更准的反转技术（相对 DDIM ） DDIM
    会 “丢细节”（如刺绣的针脚）；而 ReNoise 能让结果更贴近图片本身的细节。
    提取 SDXL 中自注意力层的输出特征
    通过 ReNoise 转换为 SDXL 能处理的 “加噪潜变量（latent vector）”（记为(z_emb)（刺绣图潜变量）、(z_des)（设计图潜变量））

  2. 特征对比：计算“刺绣特征”与“设计特征”的余弦相似度。SDXL 的核心是 UNet 结构，其中的自注意力层负责捕捉图像的空间结构与高频细节（刺绣风格恰好是高频细节）。我们需要提取对 “刺绣潜变量” 和 “设计潜变量”，分别得到每个自注意力层i的输出特征：刺绣对应的特征、设计对应的特征

    • 相似度≈1：该层输出一致，负责内容（形状、颜色）。
    • 相似度很低：该层能区分二者，负责风格（精细结构）。

  3. 模块筛选：计算特征相似度，筛选负责风格的模块通过余弦相似度衡量 “刺绣特征” 与 “设计特征” 的差异，相似度≈1：说明该层对 “刺绣图” 和 “设计图” 的特征输出几乎一致→该层负责内容（形状、颜色）；相似度很低：说明该层能区分 “刺绣图” 和 “设计图”（捕捉到了刺绣的针脚细节）→该层负责风格（精细结构）。筛选结果：对 SDXL 所有自注意力层计算相似度后，发现这 4 个模块的相似度最低：

    • 优势：从 SDXL 中筛选出专门捕捉 “刺绣风格” 的模块，避免后续训练无关模块；
      实现风格 - 内容分离：后续仅对这些模块训练 LoRA，既保证风格捕捉的准确性，又不干扰内容（形状、颜色）的生成；仅微调筛选出的模块，大幅降低训练成本与时间提升训练效率。

3.3 两阶段EmoLoRA训练

第一阶段：LoRA Block Modulation（块调制）

目标：初步分离“内容”和“风格”。


步骤：

  • 模块分割：将EmoLoRA模块分为两部分。
    • 全模块：负责“还原设计图的内容（形状、颜色）”。
    • 4个选中模块：负责“给内容加上刺绣风格（针脚、材料）”。
  • Step 1：更新整个EmoLoRA，还原内容。
  • Step 2：仅更新4个刺绣模块，添加风格。

问题：第一阶段后，4个刺绣模块虽学会了风格，但仍混杂了Step 1学到的内容信息（如参考是黄色猫咪，生成的绿色青蛙也会偏黄）。

补充数据生成：因为第一阶段只有 1 组参考数据，
模型容易过拟合，所以需要生成更多样的训练数据：
用第一阶段训练好的 EmoLoRA，
生成 N（文中10） 张新刺绣图，
筛选风格和参考像的图片5张；
生成这些新刺绣图对应的设计图。
筛选内容和参考差别大的（前 2-3 张）
给第二阶段提供更多样的训练素材。

第二阶段：Contrastive Learning（对比学习）

目标：彻底分离“风格”和“内容”。

步骤：

  • Step 1 & 2：同第一阶段，同时使用参考对和补充对训练。
  • Step 3：引入对比损失 LconL_{con}Lcon​，计算纯风格特征（纯风格 = 风格 + 内容 - 内容）。
    • 损失函数逻辑：

      Lcon=−log⁡exp⁡(s(zstyleref,zstylegen)/τ)exp⁡(s(zstyleref,zcontentgen)/τ)+exp⁡(s(zcontentref,zstylegen)/τ)L_{con} = - \log \frac{\exp(s(z_{style}^{ref}, z_{style}^{gen}) / \tau)}{\exp(s(z_{style}^{ref}, z_{content}^{gen}) / \tau) + \exp(s(z_{content}^{ref}, z_{style}^{gen}) / \tau)} Lcon​=−logexp(s(zstyleref​,zcontentgen​)/τ)+exp(s(zcontentref​,zstylegen​)/τ)exp(s(zstyleref​,zstylegen​)/τ)​

      • 分子：参考纯风格与生成纯风格的相似度，尽可能大（风格要像）。
      • 分母：纯风格与纯内容的相似度，尽可能小（风格不能像内容）。

3.4 Model Inference（模型推理）


  1. 文字输入场景：执行标准的 “文生图” 流程SDXL+EmoLoRA 根据文字生成图像，同时θe给图像加上参考刺绣风格；
    输出：文字描述内容对应的刺绣图

  2. 图片输入场景：

    • 先用 SDEdit 工具：给输入图片加噪音
    • 使用SDXL + EmoLoRA去噪还原。
    • SDXL 基础模型冻结不动+ 训练好的 EmoLoRA 4 个模块；
    • 把加噪图还原成 “带刺绣风格的图”

若有输入设计图
先启用 ControlNet-Tile保证生成图和输入设计图的 颜色一致
再判断是否需要 严格对齐边界
若是：启用 ControlNet-Canny+ 颜色校正
若否跳过这一步。

4. 结论分析

4.1 对比实验



指标包括：HFRD（刺绣风格相似度）、LPIPS（设计内容保留度）、Histogram Loss（颜色一致性）、CLIP-Score。
实验证明，该方法在刺绣定制任务上的综合表现显著优于现有风格迁移/刺绣合成方法，且在用户感知层面获得了超80%的偏好。

4.2 消融实验

  • Ours：完整方法，效果最佳。
  • w/o Contrastive Learning：不用对比学习，风格与内容存在纠缠。
  • w/o Modulation：用全部模块，缺乏针对性，效果下降。
  • 2-Block Modulation：仅用2个刺绣模块，风格捕捉能力不足。

5. 结论与局限

5.1 结论

  1. 创新性：通过“成对数据构建 + SDXL模块筛选 + 两阶段对比学习”，首次实现了仅用1张参考图精准捕捉刺绣精细风格，并彻底解耦风格与内容。

  2. 有效性：实验证明，该方法在刺绣定制任务上的综合表现（风格相似度、内容保留度、颜色一致性）显著优于现有风格迁移 / 刺绣合成方法，且在用户感知层面获得了超 80% 的偏好。

  3. 实用性：可直接接入刺绣工作流，辅助商家生成预览图、缩短生产周期。同时具备良好的泛化能力。

5.3 局限性

  1. 参考风格复杂度限制：若参考刺绣包含多种混合材料（如同时有亮片、毛绒、珠绣），方法对部分精细纹理的捕捉可能不够全面。
  2. 输入内容复杂度限制：当输入设计图/文字包含极精细细节（如复杂文字排版、微小图案），内容保留的精准度可能下降。
  3. 计算资源需求：两阶段训练虽仅微调LoRA，但仍需一定计算资源。
  4. 风格适用性：该方法针对“刺绣类高频纹理风格”设计，迁移到3D纹理、水彩等差异较大的风格时，效果可能不如专门的方法。
