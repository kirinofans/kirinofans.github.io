[论文笔记] DCT-Net:基于域校准纹理网络的肖像风格化

paper:DCT-Net: Domain-Calibrated Translation Network for Portrait Stylization
Comments: Accepted by SIGGRAPH 2022 (TOG). Project Page: this https URL , Code: this https URL
Subjects: Computer Vision and Pattern Recognition (cs.CV)

摘要


肖像风格化旨在将自然人的外貌转化为更具创意的诠释，呈现期望的视觉风格，同时保持个人特征。它在艺术创作、动画制作和虚拟化身生成等领域具有巨大的潜在应用价值。然而，创作艺术肖像画受技能限制，并且在图像创作和布局方面需要大量的人力。

本文提出了 DCT-Net，核心目标是仅用 100 张左右 的风格样本，实现“高质量、强鲁棒、广适用”的肖像风格化。该方法通过内容校准、几何扩展和纹理翻译三个模块，解决了少样本场景下的过拟合、姿态单一及细节丢失问题。

1. 研究背景与目标

1.1 肖像风格化的定义与价值

  • 定义：肖像风格化是数字艺术的重要组成部分，其目的是将自然人的外貌转化为更具创意的诠释，呈现出期望的视觉风格，同时保持个人特征。
  • 应用场景：艺术创作、动画制作、虚拟形象、APP等。
  • 痛点：创作艺术肖像画受技能限制，并且在图像创作和布局方面需要大量的人力。

1.2 核心目标

用 100 张左右 的风格样本，实现“高质量、强鲁棒、广适用”的肖像风格化。

2. 相关工作

2.1 Neural Style Transfer (神经风格迁移)

知乎：Neural Style Transfer 风格迁移经典论文讲解与 PyTorch 实现

  • 原理：神经风格迁移的工作原理基于深度学习中的卷积神经网络（CNN）。神经风格迁移使用CNN来提取内容图像和风格参考图像的特征。内容图像通常通过CNN中的较深层来捕捉，因为这些层能更好地编码图像的高级语义信息。风格图像则通过CNN的较浅层来分析，这些层能够捕捉到图像的纹理、颜色和局部模式等风格特征。通过计算内容图像和风格图像在相应层的特征表示之间的差异，可以构建一个损失函数，该函数衡量了生成图像与目标内容和风格的相似度。

  • 特征提取：

    • 内容图像：通过CNN中的较深层捕捉，编码高级语义信息。
    • 风格图像：通过CNN的较浅层分析，捕捉纹理、颜色和局部模式等风格特征。

  • 机制：计算特征表示差异构建损失函数。深层网络包含抽象内容信息，浅层网络包含像素级信息，可用于原始图像内容重构。

2.2 GAN (生成对抗网络)

  • 组成：生成对抗网络其实是两个网络的组合：生成网络（Generator）负责生成模拟数据，最终目的是“骗过”判别器；判别网络（Discriminator）负责判断输入的数据是真实的还是生成的，目的是找出生成器做的“假数据”。
    • 生成网络：负责生成模拟数据，最终目的是“骗过”判别器。
    • 判别网络：负责判断输入的数据是真实的还是生成的，目的是找出生成器做的“假数据”。

2.3 Image-to-Image Translation (图像到图像翻译)

  • 代表方法：CycleGAN 等。
  • 特点：比神经风格迁移更灵活，且是无监督方法。
  • 局限：数据依赖严重，泛化能力差。

2.4 GAN Inversion

  • 原理：基于已训练 GAN 的 latent space 编辑真实图像，且无需任何临时监督或昂贵的优化。将真实图像映射到 latent space 后，可以沿着特定方向改变 latent code 来编辑属性。
  • 局限：存在内容丢失严重、泛化能力差、依赖逆映射算法等问题。

3. 方法描述

DCT-Net 框架由 3 个模块组成：内容校准网络（CCN）、几何扩展模块（GEM）、纹理翻译网络（TTN）。

3.1 框架总览

  1. 内容校准网络（CCN）：借助预训练模型的先验知识，校准少样本目标域的“内容分布”。
  2. 几何扩展模块（GEM）：扩展源分布和目标分布的几何维度，为后续转换提供不同尺度和旋转角度的几何对称特征。
  3. 纹理翻译网络（TTN）：基于校准+扩展后的样本，以无监督方式学习跨域对应关系，实现像素级精细风格转换。

3.2 内容校准网络


目标：校准少样本目标域的偏斜分布，借助预训练 StyleGAN2 的源域分布先验，生成内容对称的源域/目标域样本。

步骤详解：

  1. 初始化目标域生成器：
    • 继承预训练 StyleGAN2 的源域分布先验。
    • 预训练完成的源域生成器 GsG_sGs​。
    • 复制权重，得到目标域生成器 GtG_tGt​。
  2. 微调 GtG_tGt​：
    • 目标：让 GtG_tGt​ 生成符合目标风格 + 保留源域内容的样本，同时校准目标域分布。
    • 损失约束：
      • GAN 损失：用判别器 DtD_tDt​ 判别生成的风格样本和真实的风格样本。
      • 内容约束 RidR_{id}Rid​：保留人脸身份。
  3. 层融合 + 样本生成：
    • 输入随机编码 zzz。
    • 层融合：将 GsG_sGs​ 的前 kkk 层与 GtG_tGt​ 的对应层融合（融合后能保留更多源域的人脸结构细节）。
    • 对称样本生成：生成源域样本（真实人脸风格，结构完整）和目标域样本（目标风格）。

3.3 几何扩展模块


背景：源域（FFHQ 数据集）的真实人脸是对齐到标准面部位置的（五官都在固定区域），这会让网络过度依赖位置语义。一旦遇到真实场景中旋转、缩放的图像（侧脸、仰头），就会容易因位置不符合训练习惯而失效。

目标：为网络注入尺度与旋转不变性，使其能适配真实场景中几何形态多样的图像，同时支持后续的全身图像推理。

方法：对上一步骤所有输入样本，加统一的随机几何变换，包含两个维度：

  • 随机缩放。
  • 随机旋转。

效果：打破位置依赖，提升鲁棒性。

3.4 纹理转换网络

目标：在无监督的条件下，学习跨域对应关系，实现像素级精细纹理转换。

损失函数：TTN 的训练是 4 个损失的加权组合：

Ltotal=λconLcon+λperLper+λtvLtv+LstyL_{total} = \lambda_{con} L_{con} + \lambda_{per} L_{per} + \lambda_{tv} L_{tv} + L_{sty} Ltotal​=λcon​Lcon​+λper​Lper​+λtv​Ltv​+Lsty​

其中，$$\lambda_{con}=2\times10^2$$、$$\lambda_{per}=1$$、$$\lambda_{tv}=10^4$$。

  1. 风格损失 LstyL_{sty}Lsty​：
    • 受相关研究中表示分解方法的启发，通过纹理和表面分解，提取风格表示。
    • 通过惩罚真实风格化图像和生成图像的风格表示分布之间的距离来计算。
  2. 内容损失 LconL_{con}Lcon​：
    • 用预训练 VGG16 网络。
    • 最终计算源图的 VGG 特征与生成图的 VGG 特征之间的 L1 距离。
  3. 面部感知约束：
    • 动机：普通损失只能约束整体风格/内容，但风格化常需要局部结构夸张变形（如卡通）。
    • 方法：基于 PatchGAN 的特征提取器，顶部接 3 个回归头（对应 3 个表情参数：左右眼和嘴巴开合度）。
    • 计算：从源图中提取面部关键点 ppp，计算左眼/右眼/嘴巴的高宽比，得到对应的 α\alphaα。用 RexpR_{exp}Rexp​ 预测生成图的表情参数，计算其与源图表情参数的 L2 距离。

全图渲染策略：

  • 与以往的研究Kim等人，2020；Song等人，2021是独立处理人脸和背景。首先从输入图像中提取对齐的人脸，并逐一进行风格化处理。然后，使用一些专门的算法渲染背景图像，并将其与风格化后的人脸融合，得到最终结果。
    本文没有采用这种复杂的流程，而是通过一次评估直接从全图渲染出风格化结果。通过实验发现，分辨率低于2K×2K的图像能够被很好地处理，而且合成的图像不会出现模糊。

  • 本文方法：通过一次评估直接从全图渲染出风格化结果。

  • 效果：分辨率低于 2K×2K 的图像能够被很好地处理，合成的图像不会出现模糊。

4. 实验与结果分析

4.1 定性与定量对比

  • 内容保留：输入图戴眼镜、有发丝细节，DCT-Net 能精准保留。
  • 风格忠诚：生成图的线条、色调和目标风格样本基本一致。
  • 定量指标：
    • FID：衡量生成图和真实风格图的相似度。
    • ID：衡量身份保留情况。
  • 用户研究：40 个受试者，100 组测试用例。
    • 82.6% 的人觉得 DCT-Net 风格最好。
    • 90.5% 的人觉得内容保留最好。
    • 结论：主观感受和客观数据一致。

4.2 消融实验

  • 去掉 CCN（w/o CCN）：存在风格失真。
  • 去掉 GEM（w/o GEM）：对于复杂姿态处理差，全身翻译效果变差。
  • 去掉 TTN（w/o TTN）：会导致内容丢失严重。

模块作用总结：

  • CCN：解决“少样本过拟合”，补全风格分布。
  • GEM：解决“位置约束”，支撑复杂姿态和全身翻译。
  • TTN：解决细节丢失，实现像素级精细翻译。

5. 结论与局限

5.1 结论

  1. 方法创新：提出了 DCT-Net，为少样本图像翻译提供了“先校准分布，再做翻译”的新方法。
  2. 性能优越：在仅 100 张左右风格样本的少样本场景下，实现“高保真、强鲁棒、多风格适配”的目标，定量指标和用户偏好度均远超现有方法。
  3. 泛化能力：实现仅用头部训练样本就能直接处理全身图像的能力。
  4. 应用价值：实践上降低了肖像风格化的数据集需求，拓展了应用场景。

5.2 局限性

  1. 极端场景失败：极端场景存在失败案例。
  2. 感知约束限制：面部感知损失的适配范围有限，不能根据用户需求调整。
  3. 实时性与分辨率：实时性与高分辨率适配不足。
  4. 交互控制：缺乏交互式控制能力。
  5. 跨大类风格适配：仅测试了“同大类风格”（如 3D 卡通、手绘、动漫都属于“具象艺术风格”）；未验证“跨大类风格”（如从 3D 卡通直接切换到抽象艺术、梵高油画风）的适配效果。
  6. 样本依赖：风格样本的多样性依赖未解决。模型性能仍受限于初始风格样本的质量，如果 100 张风格样本中缺少某类细节，模型生成这类细节时仍可能出错。
