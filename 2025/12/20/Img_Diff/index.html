[论文笔记] Img-Diff: 让MLLMs看清细节差异

paper:Img-Diff: Contrastive Data Synthesis for Multimodal Large Language Models

Comments: 22 pages, 10 figures, 16 tables
Subjects: Computer Vision and Pattern Recognition (cs.CV); Artificial Intelligence (cs.AI)

摘要

由 GPT-4V 代表的多模态大语言模型（MLLM）已成为新的研究热点，展现出惊人的涌现能力。然而，现有视觉指令微调数据集存在三个关键问题：只单图理解不跨图对比、只描述整图差异无精准局部指向、人工标注成本高规模有限。

本文提出了一套全自动的数据合成方法，无需人工标注，自动生成“相似图像对 + 局部差异区域 + 精准差异描述”的训练数据。基于此构建了高质量的对比型数据集 Img-Diff，注重物体替换的细粒度差异识别。实验表明，使用 Img-Diff 微调主流 MLLMs 后，模型在图像差异和视觉问答任务中表现优异，甚至超过 GPT-4V、Gemini 等顶尖模型。

1. 研究背景与动机

1.1 多模态大模型（MLLM）的现状

  • 研究热点：由 GPT-4V 代表的多模态大语言模型（MLLM）利用强大的大语言模型作为“大脑”执行多模态任务。
  • 涌现能力：基于图像的写作和无需 OCR 的数学推理等能力，在传统多模态方法中罕见，指明了通往 AGI 的潜在路径。
  • 训练范式：所有主流 MLLMs 遵循“两阶段训练”：
    1. 预训练阶段：用海量数据让模型学会图文对齐。
    2. 微调阶段：用指令-响应对让模型学会 VQA 等实用能力。
  • 关键因素：性能提升依赖于模型架构创新和数据集质量提升。

1.2 现有数据集的三大痛点

  1. 只单图理解，不跨图对比。
  2. 只描述整图差异，没有精准指向局部的细节。
  3. 人工标注成本高，规模有限。

1.3 灵感与目标

  • 灵感：对比学习和图像差异描述（IDC）领域已证明“精准的差异描述”能提升识别能力，但缺乏高质量、大规模数据。
  • 目标：
    1. 提出全自动数据合成方法，解决人工成本高、规模小的问题。
    2. 构建高质量对比型数据集，注重物体替换，让模型注重局部细粒度差异识别。
    3. 验证微调后的模型在图像差异和 VQA 任务中的表现。

用 Img-Diff 微调主流 MLLMs（LLaVA-1.5、MGM、InternVL2），看模型在在众多图像差异和视觉问答任务中甚至超过 GPT-4V、Gemini 等顶尖模型

2. 相关工作


2.1 MLLMs 的发展

最近，以 GPT-4V 为代表的多模态大语言模型（MLLM）成为新的研究热点，该模型利用强大的大语言模型（LLM）作为“大脑”来执行多模态任务。MLLM 所展现的令人惊讶的新兴能力，如基于图像编写故事和无需 OCR 的数学推理，在传统多模态方法中极为罕见，这表明了通向通用人工智能的潜在路径。为此，学术界和工业界都在努力开发能够与甚至超越 GPT-4V 的 MLLM，以惊人的速度推动研究的极限。

2.2 图像差异数据集

专注于图像差异的数据集通常由成对的相似图像及描述差异的文本组成：

  • Spot-the-Diff：包含同一监控摄像头在不同时间拍摄的街景图像对。
  • CLEVRChange：描绘干净背景下几何物体的场景变化。
  • CUB-Bird：着重于自然栖息地中不同鸟类物种之间的细微差异。

2.3 图像差异描述生成（IDC）

  • 定义：对比两张相似图片，捕捉视觉差异，用自然语言描述出来。
  • 挑战：相比经典 Image Caption，它涉及两张图片的内容理解与对比，更具挑战性。
  • 应用：协助区分相似品种、自动检测监控场景变化等。

3. 核心方法

3.1 框架总览


本文提出了一套端到端的自动化流程，包含三个核心步骤：

  1. 图像对生成：生成仅替换单个物体的相似图像对。
  2. 差异区域生成器：定位具体的差异位置。
  3. 差异描述生成器：生成精准的差异描述文本。

3.2 图像对生成

目标：生成“仅替换单个物体、其他场景高度相似”的图像对。

步骤：

  1. 获取字幕：从 caption database（如 MSCOCO）中获取图像字幕。
  2. 修改描述：使用大语言模型修改描述。
    • Prompt 模板：“Here is a sentence: ‘INPUT’. Please only replace one of the objects in this sentence with another object.”（INPUT 为原始字幕）。
  3. 生成图像：用 SDXL 和 Prompt-to-Prompt 技术，分别根据“原始描述”和“替换后描述”生成两张图。背景、构图高度相似，只有替换的物体不同。

3.3 Difference Area Generator (差异区域生成器)

目标：找到两张图像中“物体替换”的具体位置。
流程：图像相似度过滤 → 图像分割 → 图像-文本匹配过滤 → 差异检测 + IoU 过滤。
详细步骤：

  1. Image Similarity Filter：避免两张图差异太大。
  2. FastSAM：对图像进行分割。
  3. Image-text Matching Filter：验证“分割后的子图”是否包含“被替换的物体”，过滤无关区域。
  4. Difference Detector + IoU Filter：
    • 计算交并比，阈值设为 0.5。
    • 根据边界框坐标裁剪子图，计算子图相似度。如果相似度足够低，说明确实有差异，边界框有效。

3.4 Difference Captions Generator (差异描述生成器)


Stage 1: Object Labeling & Filtering

  1. 对每个图像对，选“图像相似度最低”的 N 个边界框（N=5）。
  2. 用 MLLM（LLaVA-NEXT）对每个边界框生成描述。
  3. 用 BLIP 验证描述是否和区域内容一致。
  4. Captions Similarity Filter：用 CLIP 提取两个描述的文本特征，计算余弦相似度。如果分数高，说明没有差异，过滤掉。

Stage 2: Difference Captions Generating
根据边界框坐标画红色框，生成差异描述。

3.5 Data Statistics (数据统计)

  • 利用 MSCOCO 字幕生成了 11.8万 对相似图像。
  • 使用图像相似性过滤器得到 38,533 对高度相似但不完全相同的图像对。
  • 通过差异区域生成器筛选，得到 117,779 条有效边界框信息（每对图最多 5 个）。
  • 最终生成 12,688 个高质量的“物体替换”实例。
  • 使用 LLaVA 预训练数据集字幕额外生成了 34,538 个物体替换样本。

4. 实验结果与分析

4.1 基准测试结果


Spot-the-Diff 数据集：
该数据集包含展示细微物体差异的街景图像对。实验结果显示，Img-Diff 数据集能显著提升模型区分图像差异的能力（策略为只增加数据不替换数据）。

The dataset of Spot-the-Diff comprises pairs of street view images that display subtle object differences. These images are obtained by capturing scenes from fixed surveillance cameras at different time.


Image-Editing-Request 基准：
专注于图像编辑，包含图像对和编辑指令。评估时计算 BLEU, METEOR, CIDEr-D, 和 ROUGE-L 分数，模型表现优异。
The Image-Editing-Request benchmark focuses on image editing. Each instance in its dataset consists of an image pair (i.e. a source image and an edited image) and an editing instruction that describes the transformation. During the evaluation, our models are required to generate transformation descriptions for these image pairs, and we then calculate the BLEU, METEOR, CIDEr-D, and ROUGE-L scores using the models’ responses and the reference answers.

通用视觉能力：
在视觉问答（VQAv2, GQA）、感知与推理（MMBench, MM-Vet, ScienceQA, SEED-Bench）、细粒度定位（POPE）等任务上均表现良好。

4.2 数据质量评估


人工评估：
随机抽 1000 个样本，找专业标注员从 3 个维度打分：边界框差异、内容描述准确性、差异描述准确性。

  • 结果：低占比极低，说明数据集精准度高，过滤机制有效。
  • 细节：218 个差异描述属于“中等错误”，即特征描述（如颜色）错误，但核心差异正确，不影响模型学习寻找差异。

5. 结论与局限

5.1 结论

  1. 方法有效：提出了有效的对比型数据合成方法，构建了端到端自动化流程，无需人工标注即可生成聚焦“物体替换”的高质量图文数据。
  2. 性能显著：Img-Diff 数据集微调主流 MLLMs 后，模型在图像差异专项任务中表现突出。
  3. 能力迁移：微调不仅优化了寻找图像差异的能力，还能迁移到通用视觉任务。
  4. 高质量多样：人工评估显示 80% 样本差异区域框选精准，80.1% 区域描述准确，70% 差异总结无误。覆盖 1203 种物体类别、3680 种物体替换组合，兼顾常见与小众物体。
  5. 消融实验：确定了数据集构建的最优过滤阈值，为 MLLMs 数据集优化提供了可复用经验。

5.2 局限性

  1. 场景局限：缺乏复杂现实场景的样本，部分样本存在细节失真问题。
  2. 模态局限：仅验证了“图像-文本”模态的效果，未探索其在更复杂多模态任务中的迁移价值。
  3. 适配性：对模型架构的适配性未深入探索。
  4. 难度梯度：数据集中的差异难度相对统一，缺乏难度梯度。
